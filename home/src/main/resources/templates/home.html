详细教程

简单教程

			1、下载frp压缩包
			下载地址

			2、提取frpc.exe和frpc.ini这两个文件

			3、修改配置文件

			打开frpc.ini
			复制下面的代码到文件里：
			[common]
			server_addr = tunnel.jlw365.com
			server_port = 7000
			log_file = ./frpc.log
			log_level = info
			log_max_days = 3
			privilege_token = 123456789

			[HTTP]
			privilege_mode = true
			type = http
			local_ip = 127.0.0.1
			local_port = 80
			custom_domains = youname.com,www.youname.com

			[TCP]
			privilege_mode = true
			type = tcp
			remote_port = 9000
			local_ip = 127.0.0.1
			local_port = 22
			use_gzip = true
			use_encrypti = true
			4、启动frp
			在ctrl+r键启动cmd用命令行操作
			cd到在所在目录
			输入 frpc.exe -c frpc.ini
			注意：不可能关闭cmd命令窗口

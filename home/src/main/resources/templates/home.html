详细教程

简单教程

1、下载frp压缩包
下载地址：http://www.diannaobos.com/frp/
2、提取frpc.exe和frpc.ini这两个文件
3、修改配置文件
使用Notepad编辑器打开frpc.ini
清空里面的内容
复制下面的代码到文件里：
[common]
server_addr = frp.lu8.win
server_port = 7000
log_file = ./frpc.log
log_level = info
log_max_days = 3
privilege_token = frp888
[HTTP_mode]
#注意修改上方[]内的内容，不要与其他隧道名称重复
#privilege_mode特权模式
privilege_mode = true
type = http
#local_ip本地服务IP
local_ip = 127.0.0.1
#local_port本地服务端口
local_port = 80
#custom_domains将自己的域名CNAME解析至frp.lu8.win，绑定多个域名可以用,分开，如果你没有自己的域名，使用我们的自动开通域名，格式:你起的名称.frp.lu8.win
custom_domains = youname.com,www.youname.com
#pool_count = 100
[TCP_mode]
#注意修改上方[]内的内容，不要与其他隧道名称重复
privilege_mode = true
type = tcp
#remote_port外部端口，可映射TCP、UDP端口范围为：9000-65535
remote_port = 9000
#local_ip本地服务IP
local_ip = 127.0.0.1
#local_port本地服务端口
local_port = 22
use_gzip = true use_encrypti
#pool_count = 2
点击保存
4、启动frp
在Ctrl+R键启动CMD用命令行操作
CD到在所在目录
输入 frpc.exe -c frpc.ini
这时如果没有提示就说明成功了
注意：关闭黑色窗口就等于关闭frp

使用教程

windows简单教程

			1、下载frp压缩包
			下载地址

			2、提取frpc.exe和frpc.ini这两个文件

			3、登录管理后台

			访问https://manager.tunnel.kisme.org/login，
			登录管理后台(如果没有帐号请注册)，在"用户资源"->"设备管理"下,
			新增一个设备。协议选择有:HTTP,HTTPS,TCP,UDP(注意大小写)
			例如添加一个对ssh的映射:
			协议:TCP
			令牌:123
			添加一个对http的映射:
			协议:HTTP
			令牌:123

			说明:
			1 添加完后5分钟之内生效.
			2 如果非http,https，域名不用填。如果是http或者https，域名可以填写自己的域名,如www.xxx.com，
			然后在自己的域名管理将www.xxx.com域名解析到www.kisme.org:5001下，如果是https，请解析到www.kisme.org:5002下。
			3 如果http或者https中域名不填写，系统会默认生成一个域名，该域名会自动解析，用户直接访问。如分配123.tunnel.kisme.org
			即访问123.tunnel.kisme.org:5001

			添加成功后，系统全自动生成一个设备号。如:1259875643,0598759845，请记得这个设备号，下面需要用到。
			3、修改配置文件

			打开frpc.ini
			复制下面的代码到文件里：
			[common]
			server_addr = 35.165.220.128
			server_port = 5000
			log_file = ./frpc.log
			auth_token = 123   ##此处为添加设备的时候设置的令牌值

			[0598759845]
			type = http
			local_ip = 127.0.0.1
			local_port = 8080  ##本地需要映射到外网的端口

			[1259875643]
			type = tcp
			local_ip = 127.0.0.1
			local_port = 22  ##本地需要映射到外网的端口
			4、启动frp
			在ctrl+r键启动cmd用命令行操作
			cd到在所在目录
			输入 frpc.exe -c frpc.ini
			注意：不可能关闭cmd命令窗口
			5、访问
			ssh映射:地址35.165.220.128，端口8001
			http映射:http://www.xxx.com:5001  ##注意这里不是8002
			
			注意:由于资源有限，暂每个用户只能添加3个设备。建议每个设备的令牌设置成一样。
